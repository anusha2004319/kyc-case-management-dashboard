## Project Overview
Developed a KYC Case Management Dashboard to track customer onboarding and periodic review cases,
monitor escalations, manage document completeness, and analyze analyst productivity.
## Key Features
• Case pipeline tracking for onboarding and periodic review cases
• Risk classification (Low Risk, Medium Risk, High Risk)
• Escalation monitoring and overdue case tracking
• Document completeness and verification tracking
• Analyst performance monitoring
• KPI-based reporting and dashboard visualization
• Case status analysis (Approved, Pending, In Review, Escalated)
## Dashboard Metrics
• Total Cases
• Approved Cases
• Escalated Cases
• Overdue Cases
• Risk Rating Distribution
• Analyst Performance Breakdown
• Document Completion Percentage
## Corporate Entity Types Covered
• Private Companies
• Public Companies
• Trusts
• Partnerships
• LLPs
## KYC Concepts Demonstrated
• Customer Due Diligence (CDD)
• Risk-Based Customer Classification
• UBO (Ultimate Beneficial Owner) Verification
• Escalation Management
• Document Verification
• SLA Monitoring
## Tools Used
• Microsoft Excel
• Data Analysis
• Dashboard Design
• Pivot Tables
• Charts and KPI Reporting
## Outcome
Built an end-to-end KYC operations dashboard that simulates real-world compliance workflows and
supports effective case management, risk assessment, and operational reporting.
